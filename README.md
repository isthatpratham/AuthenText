# AuthenText

**AuthenText** is an advanced AI-powered plagiarism detection tool designed to help users verify the originality of their content instantly. With intelligent source matching and detailed reports, AuthenText ensures your work is authentic.

## Features

-   **Instant Analysis**: Get plagiarism results in seconds.
-   **Intelligent Source Matching**: Detects and links to original sources.
-   **File Support**: Upload `.txt`, `.pdf`, and `.docx` files for analysis.
-   **Detailed Reports**: Visual highlights and similarity scores.
-   **Modern UI**: Sleek, responsive design with dark mode support.

## Tech Stack

Built with modern web technologies:
-   [Vite](https://vitejs.dev/)
-   [React](https://reactjs.org/)
-   [TypeScript](https://www.typescriptlang.org/)
-   [Tailwind CSS](https://tailwindcss.com/)
-   [shadcn/ui](https://ui.shadcn.com/)

## Getting Started

To run this project locally:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/isthatpratham/AuthenText.git
    cd AuthenText
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```

4.  Open `http://localhost:8080` in your browser.
